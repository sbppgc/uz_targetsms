TargetSMS

Модуль позволяет отправлять СМС сообщения с помощью сервиса TargetSMS при изменении статуса заказа, как покупателям, так и менеджерам магазина (одному или нескольким).

Возможности модуля:
- Отдельные шаблоны СМС для каждого статуса.
- Для каждого статуса можно задавать 2 шаблона - для покупателя и для менеджеров.
- Отправляются только сообщения с текстом. Чтобы для определенного статуса начали отправляться сообщения, достаточно заполнить шаблон. Чтобы перестали - очистить.
- Возможность указать несколько номеров телефонов менеджеров.
- Поддержка всех вариантов записи российских мобильных номеров.
- Для персонификации сообщений, в шаблонах можно использовать переменные с данными заказа и покупателя. Список доступных переменных приведен в шаблоне.

Регистрация:
Перейти на сайт https://sms.targetsms.ru/ru/reg.html?ref=amirocms и заполнить и отправить регистрационную форму.
Дождаться, когда с вами свяжется менеджер и активирует аккаунт.


Настройка модуля:
В личном кабинете TargetSMS добавить отправителя и пополнить баланс.
Установить модуль на сайт из Амиро Маркет.
В панели администратора вашего сайта перейти в модуль настроек. Заполнить логин, пароль, имя отправителя и телефоны менеджеров.
Отредактировать шаблоны СМС.
